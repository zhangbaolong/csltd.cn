运动与社交

偏向数据可视化的移动社交app

 

课程使用软件

调研、分析图、线框图、UI、动效：figma

支持系统：windows，mac

运动与社交是一个适合于各种不同基础包括零基础的UX作品集项目教程，你将有一个属于你自己的交互设计作品集项目，该项目的最终产出为一个移动端社交app。

该项目集UX设计理论、软件使用、项目应用于一体。与此同时，在项目过程中你会学习到大量数据可视化技巧，该技巧也可以成为你今后在面试和考核过程中突出重围的杀手锏。

查看套餐

以下为老师项目案例最终展示，学生将学习到老师做项目的每一步过程并将以此类推发展出自己的产品方向和项目成果。具体项目语言以学生自己申请方向自作选择。

概览

Overview

This is a mobile based application which encourage users to work out to achieve their set goals and also let the people who are with the same goals to interact and compete with each other. The interaction feature is set there to help people to have more fun in their work out process, which also boost app usage and reliance.

设计过程

Design Process

 
 

Easy to Navigate

When first launching the app, the user would see the current hot programs. Horizontal scrolling to see more of the programs and also keep the focus point at the current tab.

Expand to more info

Once the user interested in one of the programs, he/she can click to expand the card to view more info. Reviews, ratings and other spec information will appear to help the user to know better about the program. Enrolling news would also float on the top section to encourage the enrollment.

 
 
 
 

Data Track

As one the important features of this app is data visualization. The users will easily read their workout data information using the my data tab. Calorie data of the whole week will self plot into a graph for the user to keep track of the whole week.

Keep the competition

In this app, we also use integrate with the social feature. We encourage the competition to help the users finish their goals. Users can also look for local partners to work out together as the map will show the local user info relative to the user.
